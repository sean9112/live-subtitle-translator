Tray-First Subtitle Utility

字幕控制面板

常駐在 menu bar / system tray，字幕會以獨立透明 overlay 顯示。

待機中

語言方向

目前先支援繁中（台灣）和英文互譯。

收音裝置

如果預設麥克風是靜音來源，可以在這裡直接切換輸入裝置。

字幕窗

把 overlay 當成實際字幕層，控制面板只負責設定與歷史。

即時預覽

初次啟動會下載本地模型，完成後會快取到電腦上。

按下「開始收音」後，翻譯字幕會顯示在這裡。
原文字幕會同步顯示在這裡。

診斷

顯示最近幾步執行狀態，方便快速定位是哪一段沒跑到。

最近字幕

保留最近 8 筆結果，方便你邊講邊校對。
